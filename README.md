# data-career-quest
Data Career Quest est un mini-jeu rétro développé avec Next.js où le joueur démarre comme Data Analyst, Data Engineer, Data Scientist ou Data Consultant, puis évolue en fonction de ses choix, connaissances et performances à travers des missions inspirées de vrais enjeux : qualité des données, dashboards, RGPD, IA, gouvernance, ML et stratégie.
